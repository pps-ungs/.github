## Repositorios

- **[SUPPAI](https://github.com/pps-ungs/SUPPAI)** — Optimizador de cadena de distribución (3 niveles: fabricación→distribución→venta) bajo demanda estocástica (Monte Carlo), con heurísticas Hill Climbing / Random Restart / ACO.
  - **Uso**: GUI (Tkinter) y CLI (`main.py`).
  - **Datos**: PostgreSQL + generación de datasets y scripts de experimentos/animaciones.
